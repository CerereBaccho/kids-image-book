## **Kids Image Book（こども画像図鑑アプリ）**

子どもが好きな「新幹線（ドクターイエロー・のぞみなど）」や
「車（ランボルギーニ・ランドクルーザーなど）」を安全に楽しめる、
シンプルなPWA（ブラウザアプリ）です。

広告なし・外部リンクなし・追跡なし。
子ども向けに使いやすく、親も安心できる設計になっています。

---

## **主な機能**

* 検索バーから好きなワードを入力して画像を取得
* 画像一覧（サムネイル）を大きめに表示
* 画像タップでフルスクリーンビュー
* 左右スワイプで画像を切り替え
* 画像をタップしても外部リンクは一切開かない
* 出典・作者・ライセンスを画面下に表示
* PWA対応（iPhone/Androidに追加して使える）

---

## **画像について（重要）**

このリポジトリには画像ファイルを一切含んでいません。
アプリ内で表示される画像はすべて **Wikimedia Commons** のAPIを通じて取得されています。

Wikimedia Commonsの画像は、CC BY / CC BY-SA / Public Domainなどの
**各ライセンスに基づき公開されているもののみ**が利用されています。

アプリはWikimediaが提供する以下の情報を必ず表示します：

* 出典（Wikimedia Commons）
* 作者名
* ライセンス名
* ファイル名（リンクではなくテキストとして表示）

```
このアプリは Wikimedia Commons の画像を API 経由で表示しています。
画像は本リポジトリ内には含まれておらず、それぞれのライセンスに従って表示しています。
```

forkや改変を行う場合も、画像のライセンス表記を残すようにしてください。
